## 🚀 대표 프로젝트 – Route-In

운동 커뮤니티 웹 서비스

### 주요 기능
- 실시간 채팅 + 알림
- 게시글 CRUD + 댓글 + 추천
- 러닝 코스 지도
- 주간 운동 루틴
- AI 운동 추천
- 인바디 그래프
- 출석 시스템

### 기술 스택
**Frontend**
React / MUI / React Query / WebSocket

**Backend**
Spring Boot / MyBatis

**DB**
MySQL
